Benchmarks

Benchmarks orientam, validação decide.

Os números ajudam a escolher a rota, mas o teste no repositório confirma a decisão. Compare apenas cenários equivalentes.

Leitura prática

Terminal/debug exige loop de ferramenta. Arquitetura exige julgamento. Implementação comum exige eficiência no fluxo do time.

sinal técnico
Terminal-Bench style

Como interpretar

O benchmark mede combinação agente + modelo. Use como sinal para tarefas em terminal, não como ranking universal de inteligência.

task -> agente + modelo -> validação local -> evidência no PR
