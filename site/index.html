Scaffold Benchmark é modelo + agente + ambiente. Terminal-Bench roda tarefas reais em terminal/Docker com testes; Aider mostra que modo de edição, Architect/Editor e effort mudam score e custo. Compare resultados dentro do mesmo fluxo, não só pelo nome do modelo. Bench Data O mesmo modelo muda com o agente. Compare Claude Code, Cursor CLI, Codex, Opencode e afins como sistemas completos. Dois agentes com score parecido podem ter forças diferentes em patch de repo, terminal workflow e avaliação por rubric/Q&A. Governança Agente com tool access é automação privilegiada. Antes de colocar agente em fluxo padrão, defina sandbox, least privilege, allow-list, aprovação para ações destrutivas, logs, monitoramento e kill switch. Trate o agente como operador não confiável até prova em contrário. Observabilidade Sem trace, agente vira caixa-preta. Logue cada run com modelo, prompt, arquivos tocados, tool calls, tempo, erro, tokens e custo. Isso permite debugar regressão, comparar rotas e criar SLOs reais para uso de IA no fluxo dev. Custo por tarefa Meça $/PR, tempo ativo e tokens. A escolha boa para dev não é só "maior score": precisa registrar custo por tarefa, tempo de execução, tokens de saída e cache hit. Sem telemetria por PR, o ranking pode esconder uma rota cara demais para produção. Open-weight Controle local não elimina custo de inferência. Open-weight ajuda em soberania, privacidade e customização, mas rotas locais precisam provar latência, memória, KV-cache, throughput e tokens por tarefa. Modelo aberto sem orçamento de infra vira gargalo. Contexto Contexto certo vale mais que contexto grande. Agentes performam melhor quando recebem mapa do repo, arquivos relevantes, convenções do projeto e comandos de build/test. Evite despejar o repositório inteiro: contexto irrelevante aumenta custo e piora patch. Verificação Merge só com evidência executável. Código IA-assistido deve passar por testes, revisão humana, análise estática, secret scan, dependency scan e gates de CI antes do merge. Use agente para acelerar, não para substituir evidência.
